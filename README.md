# niukb_2026

我的 2026 年知识库，主要用来做 paper reading 和知识整理。

## 目录结构

```
.
├── papers/                 # 论文阅读笔记
│   ├── reading/           # 正在阅读的论文
│   ├── done/              # 已完成的阅读
│   └── todo/              # 待读清单
├── notes/                 # 主题笔记和总结
├── resources/             # 资源文件
│   ├── pdfs/              # PDF 文件
│   └── templates/         # 模板
└── README.md              # 本文件
```

## Paper Reading Workflow

1. 将新论文放入 `papers/todo/`
2. 开始阅读时移到 `papers/reading/`
3. 完成后移到 `papers/done/`
4. 在 `notes/` 中写主题总结
